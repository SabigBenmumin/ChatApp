# Chat App
this project for cn321

## must have technology

## install project
### 1. clone repo
```
git clone https://github.com/SabigBenmumin/ChatApp.git
```
### 2. install module
for bun:
```
bun install
```
for node.js:
```
npm install
```
for yarn:
```
yarn install
```

## how run app
for bun:
```
bun dev
```
for node.js:
```
npm run dev
```
for yarn:
```
yarn dev
```
